登录--微信公众平台mp.weixin.qq.com，进入【开发--开发者工具】，选择进入【web开发者工具】
进入web开发者工具，点击【绑定开发者微信号】 ,向开发者微信号发送绑定邀请。
开发者在手机微信上接受邀请，即可完成绑定。
基本配置中，找到【服务器配置--修改配置】， 将URL / Token / EncondingAESKey对应填写至服务器配置
URl

Token
{$account['token']}
点击复制
EncondingAESKey
{$account['encodingaeskey']}
点击复制

再在基本配置中，选择【IP白名单】，修改后保存。
进入【设置-公众号设置】，选择【功能设置】，依次配置业务域名/ JS接口安全域名/网页授权域名
